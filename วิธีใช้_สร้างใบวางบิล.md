# วิธีใช้

ใช้ได้ทั้ง `macOS` และ `Windows`

ไฟล์ launcher:

- macOS: [ใบวางบิล lawson.command](/Users/bhusitt./Downloads/ใบวางบิล%20lawson.command)
- Windows: [ใบวางบิล lawson.bat](/Users/bhusitt./Downloads/ใบวางบิล%20lawson.bat)

## ต้องเอาอะไรไปอีกเครื่อง

อย่างน้อยต้องมี:

- ไฟล์ `build_full_billing_workbook.py`
- ไฟล์ launcher (`.command` หรือ `.bat`)
- ไฟล์ [requirements.txt](/Users/bhusitt./Downloads/c-d/requirements.txt)

แนะนำให้ง่ายสุดคือยกทั้งโฟลเดอร์ `c-d` ไป แล้ววาง launcher ไว้ข้าง ๆ โฟลเดอร์นั้น

## macOS

1. ติดตั้ง Python 3 ถ้ายังไม่มี
2. เปิด Terminal แล้วติดตั้ง dependency:

```bash
pip3 install -r /path/to/c-d/requirements.txt
```

3. ลากโฟลเดอร์ `c-d` หรือ `SI-xxxx` ไปวางบน `ใบวางบิล lawson.command`
4. ใส่เดือน เช่น `2026-04`

## Windows

1. ติดตั้ง Python 3 และติ๊ก `Add Python to PATH`
2. เปิด Command Prompt แล้วติดตั้ง dependency:

```bat
pip install -r C:\path\to\c-d\requirements.txt
```

3. ดับเบิลคลิก `ใบวางบิล lawson.bat`
4. ใส่ path ของโฟลเดอร์ `c-d` หรือ `SI-xxxx`
5. ใส่เดือน เช่น `2026-04`

## เปลี่ยนชื่อไฟล์

เปลี่ยนชื่อ launcher ได้ เช่น:

- `ใบวางบิล lawson.command`
- `ใบวางบิล lawson.bat`

ชื่อไฟล์ launcher ไม่กระทบการทำงาน ตราบใดที่ยังอยู่ข้าง ๆ โฟลเดอร์ `c-d` หรือ script

## ตัวอย่างรันตรง

```bash
python3 build_full_billing_workbook.py "/path/to/c-d" --month 2026-04
```

หรือบน Windows:

```bat
python build_full_billing_workbook.py "C:\path\to\c-d" --month 2026-04
```
